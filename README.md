# Member Management Web App
    - A full-stack web application where employees can securely log in/create account, manage member records, and export data to CSV. 
    - Employee passwords are hashed before being stored in the database.

## Getting Started:
    - Copy the repository
    - Install dependencies (See Installation)
    - Run `python database.py` first to create the database
    - Run `python main.py`



### Installation:
- `pip install flask`
- `pip install Werkzeug`


## Built with:
- Python
- HTML, CSS
- Javascript
- Flask
- SQLite3
- Werkzeug
